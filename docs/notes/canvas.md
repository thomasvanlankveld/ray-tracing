## Pixel manipulation

The key is to write RGB values to the correct indices of an `ImageData` object. See MDN page for an abstraction to help find the indices, and the second link for how to write.

- [Pixel manipulation with canvas - Web APIs | MDN](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API/Tutorial/Pixel_manipulation_with_canvas#the_imagedata_object)
- [HTML5 Canvas: Pixel Manipulation](http://tutorials.jenkov.com/html5-canvas/pixels.html)
